Introduction
============

JSON-RPC bindings are available via the `deltachat-rpc-client <https://pypi.org/project/nexuschat-rpc-client/>`_ Python package.
This package provides only the Python bindings and requires ``deltachat-rpc-server`` binary to be installed.
`deltachat-rpc-server <https://pypi.org/project/nexuschat-rpc-server/>`_ package provides ``nexuschat-rpc-server`` binary for Linux, Windows, macOS and Android.

RPC client connects to standalone Nexus Chat RPC server ``nexuschat-rpc-server`` and provides Python interface to it.
