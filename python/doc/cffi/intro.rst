Introduction
============

CFFI bindings are available via the `nexuschat <https://pypi.org/project/deltachat/>`_ Python package.
The package contains both the Python bindings and the Nexus Chat core.
It is provided only for Linux.

The ``nexuschat`` Python package provides two layers of bindings for the
core Rust-library of the https://nexus.chat messaging ecosystem:
low-level CFFI bindings to the C interface of the Nexus Chat core
and high-level Python bindings built on top of CFFI bindings.
